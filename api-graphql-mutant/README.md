# 1- API GRAPHQL

API gestiona la carga y consulta de datos.

## Scripts

En el directorio del proyecto descargado moverse al directorio (cd ./api-graphql-mutant), ejecutar los siguientes comandos:

### `npm i`

Una vez que descargo los archivo instale las dependencias.<br />

### `prisma generate`

Comando para generar los archivos de graphql.<br />

### `npm start`

Ejecutar la aplicación.<br />
Abrir [http://localhost:80](http://localhost:80) en el navegador..<br />

Consola de adminitración prisma:
Abrir [https://eu1.prisma.sh/facundo/api-graphql-mutant/dev](https://eu1.prisma.sh/facundo/api-graphql-mutant/dev) en el navegador..<br />
